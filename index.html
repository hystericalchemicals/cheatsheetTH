Tag Syntax What does it do? No but really, *what* does it do?
block display: block; adds line breaks above and below generally speaking, arranges elements in a column
inline display: inline; does not add line breaks either above or below use with extreme caution, especially if you want to align elements with specified margins, padding etc, because that info will be ignored, 9 times out of 10, it makes the aligned elements vanish
inline-block display: inline-block; adds a line break below an element only generally speaking, arranges elements in a row. Where the elements have text, the right hand element's text will generally appear lower than you want it to. See position.
